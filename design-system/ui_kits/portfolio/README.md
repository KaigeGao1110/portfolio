# Kaige Gao Portfolio — UI Kit

Pixel-faithful recreation of Kaige Gao's personal portfolio website.

## Stack
- React (via Babel JSX in HTML)
- No Tailwind — inline styles + CSS vars from `../../colors_and_type.css`
- Framer Motion replaced with CSS transitions + keyframe animations (no npm)
- Lucide icons as inline SVG

## Screens / Sections
1. **Hero** — full-screen intro with typewriter, orbs, cyber grid, CTA buttons
2. **About** — bio text + stat cards
3. **Projects** — 2-col card grid with modal
4. **Skills** — animated skill bars (AI & Agents | Cloud & Infra)
5. **Contact** — email/GitHub/location + footer

## Usage
Open `index.html` for the full interactive prototype.
Individual JSX components: `Navbar.jsx`, `Hero.jsx`, `About.jsx`, `Projects.jsx`, `Skills.jsx`, `Contact.jsx`
